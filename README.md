# Vadim Melnic — Portfolio

Personal portfolio site + live demos. Built with HTML, Tailwind CSS and Anime.js — single files, no build step, deployable to any static host.

**Live:** https://niktofy.github.io/Portofolio/

## What's inside

| Path | Description |
|---|---|
| `index.html` | Portfolio site — dark neon design system, scroll-reveal animations, fully responsive |
| `demos/neonhud.html` | **NeonHUD** — real-time ops dashboard demo: animated charts (Chart.js), live stat counters, rig monitoring. All data simulated in-browser, zero backend |

## Stack

`HTML/CSS` · `Tailwind CSS` · `JavaScript` · `Chart.js` · `Anime.js` · `Google Fonts (Orbitron / Exo 2)`
